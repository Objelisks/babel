babel
=====

game thing

running
=======

install [node.js](http://nodejs.org/)

run `node server.js` in command line

go to [http://localhost:8080/](http://localhost:8080/) in your web browser of choice (chrome)
press a button on your game controller (also you need a game controller)

to save files run `node chunkSaver.js` as well

controls
========
Xbox 360 Wired Controller is only thing that is tested currently, but should support any gamepad.

* left stick: movement
* right stick: swing stick
* left shoulder: toggle edit mode
* right shoulder (edit): save chunks to file
* x (edit): lower terrain
* y (edit): raise terrain
